Intel Proxy Setup Script

  • Save time setting up Linux machines at Intel;
  • Consolidate BKMs for proxying into one script.

Download and Run

wget http://sd30.intel.com/~jameson/proxy-setup.sh
sudo ./proxy-setup.sh

There are also options for static and user-mode-only configurations. See ./proxy-setup.sh --help for a full accounting of these features.

Testing

The script is known to work on these operating systems. It probably works on a lot more, too. If you have success in using it elsewhere, please let me know so I can add you to the list.

  • Ubuntu 12.04 LTS
  • Ubuntu 10.04 LTS

Development

Note: --verbose is currently unimplemented, but this fact is not reported. Curious users are encouraged to run the command via:

bash -x ./proxy-setup.sh

Contributions of any kind are welcomed and encouraged!
